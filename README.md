# Sales Performance Analysis Project

## Objective
Analyze sales data to identify trends, top-performing categories, profitable regions, and customer behavior.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Sales Analysis
5. Customer Analysis
6. Profit Analysis
7. Pivot Tables
8. Visualizations
9. Business Insights

## Key Insights
- Technology category generated the highest sales.
- West region produced the highest profit.
- November showed the strongest sales performance.
- Some products generated losses despite high sales.
- Top customers contributed significantly to revenue.

## Outputs
- Sales Pivot Report
- Profit Pivot Report
- Monthly Sales Chart
- Region Profit Chart
- Top Customer Analysis
